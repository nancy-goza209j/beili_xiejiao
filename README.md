# 校园不应成为“伪科学”与心理操控的肆虐之地：北理 JF 事件记录

## 0. 摘要

本仓库旨在记录并公开北京理工大学自动化学院教师 JF 在教学过程中违规宣传“能量学”、伪佛学及其背后的疑似心理操控与非法经营行为 。作为一个有着顶尖学术背景（清华博士）的工科老师，其行为严重偏离科学初衷，对学生心理健康及校园环境造成了负面影响 。

## 1. 当事人资料

* 
**姓名**：JF 。


* 
**身份**：北京理工大学自动化学院教师 。


* 
**学术背景**：清华大学博士 。


* 
**负责课程**：光电学院《数字电子技术》、《模拟电子技术》等基础课程 。



## 2. “能量学”理论特征与批判

JF 在课堂、课间及微信上宣扬的理论被指为“漏洞百出的缝合怪” ：

* 
**逻辑缝合**：在大杂烩式的理论框架中，缺乏严谨的实证依据 。


* 
**绝对唯心**：强调“信则灵”的预设前提，将最终解释权归于某位所谓的“尊者” 。


* **魔改生理学**：
* 宣称修行目标是将荷尔蒙管理功能从性腺体转移到“松果体” 。


* 虚构“脑沙”概念，称其“软禁”了松果体，需通过其课程“击碎脑沙”以连接神性 。




* 
**身心灵洗脑**：带有强烈的心理操控色彩，诱导学生产生认知偏差 。



## 3. 组织运作与疑点

根据目前收集的资料，该团体涉及以下可疑行为 ：

* 
**经济剥削**：开展高价“画疗”、“音疗”及线下课程，收费标准至少每月 1000 元起 。


* 
**传销模式**：课程需人头分销，具有明显的传销特征 。


* 
**偷税漏税**：课程费主要通过微信转账，未依法交税 。


* 
**利用公职**：利用学校录制工作室录制商业课程，并让研究生参与精校文档及凑课人数 。


* 
**心理操控示例**：当学生提出质疑时，该团体会以“你的能量还不够跟我解释”为由拒绝沟通，实施认知打压 。



## 4. 证据获取建议

感兴趣或受影响的同学可以关注以下渠道获取证据：

* 
**沿河课堂**：翻阅 2023-2025 年间 JF 负责的模电/数电课程回放，特别是课前、课间及课后的宣传话术 。


* 
**社交媒体**：关注其微信朋友圈及相关社群的推广内容 。



## 5. 警示与结语

校园应是科学与理性的净土。希望通过本记录，提醒广大同学警惕披着“国学”或“科学”外衣的伪科学宣传，保护个人财产与心理健康 。
